Implementation

I have implemented volfee.links in Go using http/net package and BoltBD database. You can see the implementation on my github repository.

Are volfee-links persistent?

Sadly no. I am running a script daily that cleans up the database to prevent it from overfilling and keep it free from malicious content like Illegal File Sharing or Malware. If you are interested in implementing go links for your organization, please have a look at my github repository to see how I decided to implement it.
